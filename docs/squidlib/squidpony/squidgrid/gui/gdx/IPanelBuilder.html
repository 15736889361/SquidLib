      • buildScreenWide

        SquidPanel buildScreenWide(int screenWidth,
                         int screenHeight,
                         int desiredCellSize,
                         TextCellFactory tcf)
        This method builds a panel whose number of cells is such that the cell size is close to desiredCellSize. This method only supports square cells.
        Parameters:
        screenWidth - The screen's width, in number of pixels, as given by libgdx.
        screenHeight - The screen's height, in number of pixels, as given by libgdx.
        desiredCellSize - The cell size that you would like the panel to have. Width and height cannot be different, because this method supports only square cells.
        tcf - The TextCellFactory to use. Can be null to create the panel.
        Returns:
        A screen wide squid panel, margins-aware, and with its position set.

        Builds a panel by the number of requested cells.

        cellHeight - The height of a cell (in pixels).
        tcf_ - The text cell factory to use, if any.
        Returns:
        A freshly built panel.
