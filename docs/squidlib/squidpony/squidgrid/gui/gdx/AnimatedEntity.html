      • setDirection

        public void setDirection(squidpony.squidgrid.Direction dir)
        Rotates this so that "up" points in the specified direction. Only some Actors can actually be rotated; Images can, for example, but Labels cannot. This method is most likely to be used with TextCellFactory.makeDirectionMarker(Color), TextCellFactory.makeDirectionMarker(Collection, float, boolean), or one of the directionMarker methods in SquidPanel or SquidLayers, since those produce an Image (or ColorChangeImage) that can be sensibly rotated to indicate a direction over a cell.
        Parameters:
        dir - the direction that "up" for this should point toward
