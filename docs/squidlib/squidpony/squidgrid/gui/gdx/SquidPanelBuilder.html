Class SquidPanelBuilder

  • java.lang.Object
    • squidpony.squidgrid.gui.gdx.IPanelBuilder.Skeleton
      • squidpony.squidgrid.gui.gdx.SquidPanelBuilder
  • All Implemented Interfaces:
    IPanelBuilder

    public abstract class SquidPanelBuilder
extends IPanelBuilder.Skeleton
    An almost-concrete implementation of IPanelBuilder. This class makes the assumption that font files are only available for square and even sizes. The only method to implement is fontfile(int) that must return the name of the file where a font of a given size lives.

    I think this class should be generalized so that it supports cell width and cell height that are proportional (for example: a height that is the double of the width), but not to arbitrary combinations. In that case adjustCellSize(int) would become impossible to implement.
