    public class LinesPanel<T extends com.badlogic.gdx.graphics.Color>
extends com.badlogic.gdx.scenes.scene2d.Actor
    An actor capable of drawing IColoredStrings. It is lines-oriented: putting a line may erase a line put before. It is designed to write text with a variable-width font (as opposed to SquidPanel). It performs line wrapping by default. It can write from top to bottom or from bottom to top (the default).

    This tweet shows an example. The panel at the top of the screenshot is implemented using this class (with drawBottomUp being true).

    This class is usually used as follows: 

     final int nbLines = LinesPanel.computeMaxLines(font, pixelHeight);
 final LinesPanel lp = new LinesPanel(new GDXMarkup(), font, nbLines);
 lp.setSize(pixelWidth, pixelHeight);
 stage.addActor(lp);

    Contrary to SquidMessageBox, this panel doesn't support scrolling (for now). So it's suited when it is fine forgetting old messages (as in brogue's messages area).

    An alternative, doing similar lines-drawing business, but being backed up by {@link SquidPanel}.
