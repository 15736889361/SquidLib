squidpony.squidgrid.mapping.locks

Class Edge

  • java.lang.Object
    • squidpony.squidgrid.mapping.locks.Edge

  • public class Edge
extends Object
    Links two Rooms.

    The attached Symbol is a condition that must be satisfied for the player to pass from one of the linked Rooms to the other via this Edge. It is implemented as a Symbol rather than a Condition to simplify the interface to clients of the library so that they don't have to handle the case where multiple Symbols are required to pass through an Edge.

    An unconditional Edge is one that may always be used to go from one of the linked Rooms to the other.
