Class RectangleRoomFinder

  • java.lang.Object
    • squidpony.squidgrid.mapping.RectangleRoomFinder

  • public class RectangleRoomFinder
extends Object
    An algorithm to find rectangle areas in dungeons. It is a simpler and faster alternative to RoomFinder. You can execute RectangleRoomsFinderTest to see how it performs.
    Author:
    smelC
    See Also:
    A fancier room finder
