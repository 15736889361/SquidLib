Interface ICombinedPanel<T>

  • Type Parameters:
    T - The type of colors.
    All Known Implementing Classes:
    ICombinedPanel.Impl

    @Beta
public interface ICombinedPanel<T>
    The combination of two panels, one to color the background, the other to write characters on the foreground.

    • There is a very generic implementation in this file: ICombinedPanel.Impl that you should use if you're combining generic things.
    • There is a libgdx-Group based implementation that offers more features and that you should likely use if you're a new user (in squidlib-gdx).

    smelC
