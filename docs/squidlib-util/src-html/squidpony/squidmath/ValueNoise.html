003/**
004 * Performance-oriented "white noise" generator for 1D, 2D, 3D, 4D, and 6D. Produces noise values from -1.0 inclusive
005 * to 1.0 exclusive. Implementation is based on {@link CrossHash.Wisp#hash64Alt(double[])}, multiplying the doubles
006 * given as input by medium-sized doubles (between 100 and 200, with a large section after the decimal point), then
007 * flooring them and doing the rest of the math with longs. Finishes by passing the top 52 bits as a significand to
008 * {@link NumberTools#longBitsToDouble(long)}, using an exponent that allows this to produce numbers between -1.0 and
009 * 1.0. Has special treatment for the seed when present (since it is an int).
010 * <br>
011 * This can also be used as a sort of hashing function that produces a double, if you find a need for such a thing, with
012 * {@link #hash(double...)}.
013 */
014public class ValueNoise implements Noise.Noise1D, Noise.Noise2D, Noise.Noise3D, Noise.Noise4D, Noise.Noise6D {
